# Forge2 Kanban Board

## Tech Stack
- React (Vite)
- Laravel
- SQLite

## Frontend Setup

cd frontend
npm install
npm run dev

## Backend Setup

cd backend
composer install
php artisan migrate:fresh
php artisan serve

## Live URL

https://forge2-qualifier-dipti-dm12smpuw-forge14.vercel.app

## Notes

Frontend is deployed on Vercel.
Backend runs locally using Laravel and SQLite.
